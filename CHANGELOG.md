# Changelog

## 1.0.0

### Added
- `geo/haversine.py` — Haversine formula, pairwise distance matrix, and nearest-neighbour search
- `geo/clustering.py` — greedy radius-based clustering with centroid and spread summary
- `stats/descriptive.py` — mean, median, variance, and standard deviation with Bessel's correction
- `stats/pipeline.py` — end-to-end pipeline tying distance, clustering, and statistics together
- `README.md` — project overview with cross-file links and quick start
